# Reminder App

A web application that lets you create notes with reminders and sends notifications directly to your phone via Telegram.

## Features
- User authentication (sign up / log in)
- Create and delete notes
- Set a date and time for reminders
- Receive Telegram notifications when a reminder is due

## Built with
- Python / Flask
- SQLite
- Telegram Bot API
- APScheduler
